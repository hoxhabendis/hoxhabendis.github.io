About the workshop


The participants will be able to understand the importance of Cupping therapy and how it will help them and others to heal the diseases. There are many techniques in this therapy, which will be taught to the participants.


Modules


  • What is Cupping therapy
  • Types of treatment which can be done from this therapy
  • Types of cups which we use for healing

Learning Outcomes


Participants will have the knowledge of how to apply to pressure on ears and how cupping will help to heal the pain.


Who Should Attend

Gym trainer, Housewife, Massage therapy, Acupressure therapist, Beautician, Naturopathy therapist, Yoga teacher, Doctors etc.
